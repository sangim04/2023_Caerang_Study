## 구상
인벤토리에 필요한 기능 https://rito15.github.io/posts/unity-study-rpg-inventory/   
가장 이해하기 쉽게 정리되어있었다.   

## 구현할 기능
- 아이템 추가
- 아이템 사용
- 아이템 이동
- 아이템 개수 나누기
- 인벤토리 빈칸채우기

## 아이템
- Item : 인벤토리 각 슬롯에 들어가는 실제 아이템. 각각의 아이템이 개별적으로 갖는 데이터들을 보관
  - 수량을 셀 수 있는 아이템(파밍템)
  - 장비 아이템(스킬)

## 아이템 데이터
- ItemData : 각 아이템이 공통으로 가질 데이터들을 보관하는 클래스
